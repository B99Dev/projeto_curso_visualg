# Exercícios de Algoritmos com Visualg

Repositório de estudos de lógica de programação utilizando Portugol/Visualg.

## Objetivo

Construir uma base sólida de raciocínio lógico por meio de exercícios pequenos e progressivos, antes da aplicação dos mesmos conceitos em linguagens como Java e JavaScript.

## Conteúdos praticados

- Primeiro algoritmo e variáveis
- Entrada e saída de dados
- Operadores aritméticos, relacionais e lógicos
- Estruturas condicionais
- Estruturas de controle
- Concatenação

## Como executar

1. Instale o [Visualg](https://visualg3.com.br/).
2. Abra um arquivo com extensão `.ALG`.
3. Execute o algoritmo pelo próprio Visualg.
