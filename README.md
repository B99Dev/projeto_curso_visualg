# Exercícios de Algoritmos com Visualg

Repositório de estudos de **lógica de programação** utilizando **Portugol/Visualg**.

## 📋 Sobre o Projeto

Este projeto contém exercícios práticos para aprendizado de conceitos fundamentais de programação através da linguagem Visualg, que utiliza pseudocódigo em português para facilitar o entendimento de algoritmos.

### Objetivo
Consolidar conhecimentos em:
- Lógica de programação
- - Estruturas de dados básicas
  - - Estruturas de controle
    - - Operadores lógicos e aritméticos
      - - Entrada e saída de dados
       
        - ---

        ## 📁 Estrutura do Projeto

        ```
        projeto_curso_visualg/
        ├── CONCATENA.ALG                    # Exercício: Concatenação de strings
        ├── ENTRADA E SAIDA DE DADOS.ALG     # Exercício: Input/Output
        ├── ESTRUTURA-SE-SENAO.ALG           # Exercício: Estrutura condicional if/else
        ├── ESTRUTURAS DE CONTROLE.ALG       # Exercício: Loops e controle de fluxo
        ├── MEU PRIMEIRO ALGORITMO.ALG       # Exercício: Primeiro programa
        ├── OPERADORES ARITMÉTICOS.ALG       # Exercício: Operações matemáticas
        ├── OPERADORES RELACIONAIS.ALG       # Exercício: Comparações e operadores relacionais
        ├── TIPO DE OPERAÇÕES LÓGICAS.ALG    # Exercício: Operadores lógicos (AND, OR, NOT)
        ├── VARIÁVEIS.ALG                    # Exercício: Declaração e uso de variáveis
        ├── README.md                        # Este arquivo
        ├── LICENÇA                          # MIT License
        └── .gitignore                       # Arquivo de exclusão
        ```

            ## 🚀 Como Executar

            ### Pré-requisitos
            - Baixar e instalar [Visualg](https://www.apoioinformatica.com.br/produtos/visualg/)
           
            - ### Passos
            - 1. Abra o arquivo `.ALG` desejado no Visualg
              2. 2. Clique em "Executar" ou pressione `F9`
                 3. 3. Siga as instruções do programa
                   
                    4. ### Exemplo
                    5. ```
                       Para executar VARIÁVEIS.ALG:
                       1. Abra Visualg
                       2. File → Open
                       3. Selecione VARIÁVEIS.ALG
                       4. Pressione F9 para executar
                       ```

                       ## 📚 Conteúdo dos Exercícios

                       ### Básico
                       | Arquivo | Descrição |
                       |---------|-----------|
                       | `MEU PRIMEIRO ALGORITMO.ALG` | Seu primeiro programa em Visualg |
                       | `VARIÁVEIS.ALG` | Declaração, inicialização e uso de variáveis |
                       | `ENTRADA E SAIDA DE DADOS.ALG` | Leitura de entrada do usuário e exibição de saída |

                       ### Operadores
                       | Arquivo | Descrição |
                       |---------|-----------|
                       | `OPERADORES ARITMÉTICOS.ALG` | Adição, subtração, multiplicação, divisão |
                       | `OPERADORES RELACIONAIS.ALG` | Maior, menor, igual, diferente, etc. |
                       | `TIPO DE OPERAÇÕES LÓGICAS.ALG` | AND (E), OR (OU), NOT (NÃO) |

                       ### Estruturas de Controle
                       | Arquivo | Descrição |
                       |---------|-----------|
                       | `ESTRUTURA-SE-SENAO.ALG` | Decisões: if/else |
                       | `ESTRUTURAS DE CONTROLE.ALG` | Loops: for, while, repeat/until |

                       ### Avançado
                       | Arquivo | Descrição |
                       |---------|-----------|
                       | `CONCATENA.ALG` | Operações com strings |
